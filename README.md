## Overview

This repository contains an exploratory data analysis (EDA) of a worldwide YouTube channels dataset. The goal is to:

- Understand how creators grow (subscribers, views, uploads)
- Identify category-wise and country-wise behaviour
- Study trends such as channel creation timelines and seasonal upload patterns
- Explore correlations (for example, subscribers vs. views, and population vs. subscriber counts)

The analysis is designed to be reproducible, well-documented, and easy to extend.

## What this project covers

- Data cleaning and preparation (missing values, data types, outlier handling)
- Summary statistics for subscribers, views, uploads, and estimated earnings
- Category-wise and country-wise breakdowns and comparisons
- Temporal trends: channel creation over time and seasonality in uploads
- Correlation and regression explorations for relationships such as:
  - Subscribers vs. views
  - Population vs. subscriber count
  - Upload frequency vs. subscriber growth
