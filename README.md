This project explores a worldwide dataset of YouTube channels to understand how creators grow, how categories behave, and how different country-level factors relate to YouTube presence. The analysis focuses on 20 simple questions and uses clear visuals to explain the patterns.

What this project covers
Cleaning and preparing the dataset

Understanding subscribers, views, uploads, and earnings

Looking at category-wise and country-wise behavior

Studying trends like channel creation over time and seasonal uploads

Exploring correlations such as subscribers vs. views or population vs. subscriber count

Tech stack
Python, Pandas, NumPy, Matplotlib, Seaborn
